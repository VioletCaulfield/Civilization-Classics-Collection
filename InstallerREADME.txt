======================================================================
ABOUT
=====

Civilization Classics Collection is a fan-made compatibility, preservation and accessibility project bringing together three classic Civilization releases into a single modern Windows package.

Included titles:

* Sid Meier's Civilization (Windows)
* Sid Meier's Civilization II Multiplayer Gold Edition
* Sid Meier's Civilization II: Test of Time

The purpose of this collection is simple:

Do the hard work once, so players don't have to.

Over the past three decades, these games have accumulated numerous unofficial patches, compatibility fixes, codecs, installers, community enhancements and workarounds required for reliable operation on modern Windows systems.

Rather than requiring players to locate and install these individually, Civilization Classics Collection packages everything into a single installer with a unified launcher featuring integrated mod management, profiles, metadata editing and modern compatibility enhancements.

The project originally began as a personal effort to make Civilization work reliably for a family member on modern hardware and has gradually evolved into a broader preservation and compatibility project.

======================================================================
LEGACY MOD COMPATIBILITY
========================

Many classic Civilization II modifications were distributed using inconsistent archive layouts.

The Mod Manager automatically handles numerous legacy archive structures including:

* Nested content folders
* Legacy deployment layouts
* Extra_Files directories
* 1Extra_Files directories

Legacy auxiliary folders are automatically merged into the appropriate deployment location during import.

Executable files contained within mod archives are ignored and are never deployed over installed game executables.

======================================================================
AUTOMATIC BACKUP & RESTORATION
==============================

Before deploying any modification, original game files are automatically backed up.

When the game closes, all original files are restored automatically.

Players can freely experiment with graphics mods, scenarios and total conversions without permanently modifying their installation.

======================================================================
PREREQUISITES
=============

Where required, the installer automatically detects and installs supported Microsoft runtime components.

These include:

* Microsoft .NET Desktop Runtime
* Microsoft Visual C++ Runtime

Additional compatibility components include:

* LAV Filters
* Legacy Indeo Video codecs
* Optional Microsoft DirectPlay support

These components are included solely to improve compatibility and reduce manual troubleshooting.

======================================================================
TEST OF TIME MINI ADDON
=======================

The official Civilization II: Test of Time Mini Addon was originally distributed using a legacy 16-bit InstallShield installer.

Because modern Windows systems cannot execute the original installer directly, the installation payload was extracted, verified and incorporated into the collection.

The included scenarios have been tested and function as expected through the integrated launcher.
